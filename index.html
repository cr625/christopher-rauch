Projects

ProEthica

An LLM-based ethical decision-making framework incorporating domain ontologies and role-based ethics for precise reasoning about entities, relationships, and professional guidelines.

Using domain ontologies to create structured knowledge, ProEthica evaluates decisions through professional roles bound by specific guidelines. It combines ontological reasoning with LLM capabilities via Model Context Protocol to ensure valid input/output structures and enhance ethical reasoning with structured domain knowledge.
